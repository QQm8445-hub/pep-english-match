### 2.2 消消乐核心玩法

**棋盘规则：**
- 5×5 网格，共 25 格
- 从当前单元单词库随机填充（允许重复）
- 每局独立随机，保证可玩性

**消除规则：**
1. 点击第一张卡片 → 进入「选中」状态（高亮边框）
2. 再次点击**同一格** → 取消选中（不消耗步数）
3. 再次点击**相同单词** → 消除棋盘上所有该单词，步数-1，分数+（消除数×5），播放发音
4. 再次点击**不同单词** → 切换选中目标，步数-1

**步数与分数：**
- 每局限制 **25 步**
- 消除一个单词 **+5 分**
- 道具消除：炸弹+10分 / 火箭+8分/格 / 魔力球+12分/格

**胜负判定：**
- 分数 ≥ 150 分 → **通关** 🎉
- 步数 = 0 且分数 < 150 → **失败** 😢

### 2.3 游戏道具系统

| 道具 | 图标 | 效果 | 计分 | 步数 |

| 炸弹 | 💣 | 随机消除 1 个方块 | +10 | 消耗1步 |

## 五、UI 设计规范

### 配色方案
| 元素 | 颜色 |
|---|---|
| 首页背景 | 线性渐变 `#87e8ff → #fff5cc` |
| 棋盘背景 | 金黄渐变 `#ffd966 → #ffcc00` |
| 主按钮 | 橙红渐变 `#ff6b6b → #ff8e53` |
| 次按钮 | 白底红字，浅红边框 |
| 年级圆形 | 三年级红/四年级青/五年级紫/六年级橙 |

### 字体规范
| 场景 | 大小 |
|---|---|
| 单词卡片 | `clamp(11px, 3vw, 15px)`，加粗 |
| 标题 | `clamp(26px, 6vw, 36px)` |
| 按钮 | `14px`，加粗 |

### 动效规范
| 动效 | 描述 |
|---|---|
| 消除动画 | 先放大至1.3x，再缩小至0，0.4s |
| 选中效果 | scale(1.08) + 橙红色边框光晕 |
| 按钮hover | translateY(-2px) + 加深阴影 |
| 页面切换 | opacity + scale(0.95)，0.25s |
| 进度条 | 分数变化时 width 过渡，0.4s |

### 响应式
- 棋盘最大宽度 380px，自适应小屏
- 字体使用 `clamp()` 自动缩放
- 按钮宽度 100% 填充
