# 🐣 人教PEP版英语消消乐

> 面向小学3～6年级的英语单词消消乐教育游戏，完全同步人教PEP版教材内容。

## 🎮 游戏特色

- 📚 同步人教PEP 3～6年级全册单词
- 🎯 消消乐玩法，寓教于乐
- 🔊 标准美式英语发音
- 📱 手机、平板、电脑全屏幕自适应
- 💾 无需服务器，本地运行

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可游玩，无需安装任何依赖。

## 📁 项目结构

```
pep-english-match/
├── index.html          # 主游戏文件（单文件版）
├── README.md           # 项目说明
└── docs/
    └── mvp-spec.md     # MVP需求文档
```

## 🗺️ 版本规划

- **v0.1 MVP** - 三年级Unit1，核心消消乐玩法 ✅
- **v0.2** - 三年级全册Unit1～6
- **v0.3** - 游戏道具（炸弹/火箭/魔力球）
- **v1.0** - 3～6年级完整内容 + 错题本 + 家长统计

## 📖 教材版本

人教版PEP（People's Education Press）小学英语 3～6年级

## 🛠️ 技术栈

- HTML5 + CSS3 + 原生JavaScript
- Web Speech API（单词发音）
- localStorage（本地数据存储）
